    grid

    @mixin grid($selector, $listColumns, $gutter) { ... }

    Description

    mixin para crear cuadriculas

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $selector

    nombre del selector css de cada item de la cuadricula.

    		Element—none
    $listColumns

    lista de numeros de columnas en cada breakpoint. Ej: (1,2,3,4), el numero de columnas del ultimo breakpoint se hereda a breakpoints superiores

    		List—none
    $gutter

    el espaciado entre cada elemento de la cuadricula. Por defecto es de 1em.

    		Number—none

    Example

    .gallery {
  @include grid(li, 5 );
}

    edItem

    @mixin edItem($s, $m, $l, $xl) { ... }

    Description

    mixin para crear un ed-item Description Los argumentos que acepta son las diferentes medidas que tendra el elemento en diferentes resoluciones de pantalla. Las medidas pueden especificarse como porcentajes ( % ) o como fracciones (1/3, 3/3 o 3/6, 2/6).

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $s

    para resoluciones entre 0px y 640px

    		Number—none
    $m

    para resoluciones entre 640px y 1024px

    		Number—none
    $l

    para resoluciones entre 1024px y 1440px

    		Number—none
    $xl

    para resoluciones desde 1440px en adelante.

    		Number—none

    Example

    .container {
  @include edContainer;
}

.content {
  @include edItem(100, 75, 60);
}
.sidebar {
  @include edItem(100, 25, 40);
}

    Author

    • Alvaro Felipe

    fromTo

    @mixin fromTo($from, $to) { ... }

    Description

    fromTo la regla se ejecuta entre ambos breakpoints.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $from

    breakpoint desde que medida se iniciara la regla.

    		Number—none
    $to

    breakpoint hasta que medida se ejecutara la regla.

    		Number—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .elemento {
  background-color: cyan;
  @include fromTo(s, l) {
    background-color: darken(cyan, .8);
  }
}

    Throws

    • El segundo argumento del mixin fromTo debe ser mayor que el primero

    • El segundo argumento del mixin fromTo() debe ser un breakpoint del Core (#{map-keys($breakpoints)}) o un número en px, em o rem

    • El primer argumento del mixin fromTo() debe ser un breakpoint del Core (#{map-keys($breakpoints)}) o un número en px, em o rem

    • Los argumentos del mixin fromTo() deben ser breakpoints del Core (#{map-keys($breakpoints)}) o números en px, em o rem

    • [function] isCoreBreakpoint
    • [function] getBp

    offset

    @mixin offset($s, $m, $l, $xl) { ... }

    Description

    mixin para aplicarle un desplazamiento a un elemento. Description el unico parametro necesario es el primero, luego los demas parametros si no estan especificados toman la medida del anterior.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $s

    desplazamiento para medidas menores a 640px.

    		Number—none
    $m

    desplazamiento para medidas mayores a 640px y menores o iguales a 1024px.

    		Number—none
    $l

    desplazamiento para medidas mayores a 1024px y menores o iguales a 1440px.

    		Number—none
    $xl

    desplazamiento para medidas mayores a 1440px en adelante.

    		Number—none

    Example

    .sidebar {
  @include offset(2, 5, 7);
}
